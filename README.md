# Autonomous Sumo Robot (SASIMO)

A robotics project developed for a university robotics course using ESP32, BTS7960 motor driver, and JGB DC motors.

## Features

- Wireless robot control
- PWM motor speed control
- Differential drive movement
- High-torque motor system
- Stable pushing performance for sumo competition

## Technologies Used

- ESP32
- BTS7960 Motor Driver
- JGB DC Motor
- Arduino IDE
- Embedded Systems

## Project Purpose

This project was developed to explore robotics systems, embedded control, and motor driver integration for competitive robot applications.
